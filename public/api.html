Image APIs

Web Screenshoter

Take high quality website screenshots with this API!

/image?url=https://google.com

Image Inverter

Invert the colors of an image with this simple API!

/invert?url=https://poopoo-api.vercel.app/images/example.png

Grayscale Image

Grayscale an image with this simple API!

/grayscale?url=https://poopoo-api.vercel.app/images/example.png

Pixlate Image

Pixelate an image with this simple API!

/pixel?strength=10&url=https://poopoo-api.vercel.app/images/example.png

Number/Text APIs

UUID generator

Randomly gererate a UUID, it will always be different!

/api/uuid

Roman Numerals Converter

Convert Roman numbers into normal intergers or do the oposite!

/api/roman

Remove Emojis/Characters

Remove all emojis from text | Give a list of emojis | Remove every character except letters and numbers

/api/remove

Emojify text for discord

Turns inputed text to emojis

/api/emojify

Flatten Lists

Simplifying a list by removing all lists in it!

/api/flat

Password Generator

Generate a random password, it will always be different everytime!

/password

Binary Encoder

Encode any piece of text into binary numbers! *01101000 01101001*

/encode?text=text

Binary Decoder

Decode binary numbers into normal text / decimal numbers *hi*

/decode?binary=binary

Annoying Discord Spoilers

Wow, these spoilers are so annoying!

/spoil?text=text
